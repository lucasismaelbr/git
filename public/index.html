Principais comandos do git

  • git clone https://siteshowdebola.git
    (clona o repositorio)
  • git commit nome_do_arquivo -m "Conteúdo do commit"
    (Lança um commit na copia da máquina fisíca)
  • git push
    (Lança todos os arquivos que já foram commitados e adicionado para a nuvem)
  • git rm nome_do_arquivo
    remove arquivos
  • git log
    Puxar todas ás logs de alterações, commits e etc
  • git log --oneline
    Puxar todas ás logs de alterações, commits e etc resumidamente e poucas linhas
  • git status
    Faz uma verificação para ver se seu arquivo é o mais recente e se ele já sofreu alterações
  • git add nome_do_arquivo
    Adiciona um arquivo ao git
